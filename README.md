# Todo list

Ссылка на проект из каталога: https://github.com/dwyl/javascript-todo-list-tutorial

Веб-сайт для управления списком задач и сохранение языка. С сортировкой задач по активным и выполненным, с возможностью отметить все.

# Клонировать репозиторий
```sh
git clone https://github.com/AndreiGapon/pptest
cd javascript-todo-list-tutorial
```
Запустить сервер
```sh
npm start
```
